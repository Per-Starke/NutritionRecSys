Help us build a solid foundation of ratings as the database for this Recipe Recommender

Please, click the button below to get started.
On the page showing up, there is a button linking to a recipe.
Click this button, and a detailed description of the recipe will show up in a new tab. Carefully look at the provided image, read the ingredients, description and all provided information, and based on that, at the bottom of the recipe-page, give a rating from 1 to 5 stars for this recipe.
Once you rated the recipe, close the tab and go back to the tab with the button linking to this recipe.
Reload that page, and the button will now link to a new recipe. Rate that recipe, reload the page containing the link, and so on, for as many recipes as possible. Thanks a lot for your help!
